SAMDECalibrate 

Function used by SAMDE type that implements the SaMDE genetic algorithm to calibrate a model according to a fit function, it returns a table with {fit = the best fitness value, instance = the instance of the best model, generations = number of generations it took to the genetic algorithm reach this model}.

Arguments

  • #1: Table containing the model parameters.
  • #2: model The model that will be calibrated by the function.
  • #3: fit() A function that receive a model as a parameter and determines the fitness value of such model.
  • #4: maximize An optional paramaters that determines if the models fitness values must be maximized instead of minimized, default is false.
  • #5: size Determines the size of the populations used in the SaMDE algorithm (recommended size: (10*dim)).
  • #6: maxGen If a model generation reach this value, the function stops.
  • #7: threshold If a model fitness reach this value, the function stops.
  • #8: Optional seed parameter for Random function, default value is os.time().

Usage


local fit = function(model, parameters)
	local m = model(parameters)
	m:execute()
		return m.result
end

local best = SAMDECalibrate({x = Choice{min = 1, max = 10, step = 2}, finalTime = 1}, MyModel, fit(), false, 30, 100, 0)
