SAMDE

Type to calibrate a model, returns a SAMDE type with the fittest individual, the fit value and the number of generations.

Arguments

  • fit: A function that receives a Model and calibrated parameters to calculate the instance fitness value.
  • maxGen: If a model generation reach this value, the function stops.
  • maximize: An optional paramaters that determines if the models fitness values.
  • model: A model.
  • parameters: A table of parameters to be calibrated.
  • seed: Optional seed parameter for Random(), default is system time. must be must be maximized instead of minimized, default is false.
  • size: the population size for each generation.
  • threshold: If a model fitness reach this value, the function stops.
